A terminal-first fantasy adventure inspired by the legendary **Zork I, II, III** experience.
Play in the dark, map the unknown, collect strange objects, solve puzzles, and survive what waits in the underground empire.

## The Adventure

- 3 campaigns selectable at startup: **Zork I, Zork II, Zork III**
- Mobile-first terminal UI (perfect for narrow Termux screens)
- Full terminal compatibility on **Termux, Linux, macOS**
- Multi-language gameplay: **English, Italiano, Espanol**

## Install

### NPM (Termux)

```bash
pkg update && pkg install -y nodejs-lts
npm i -g @mmmbuto/zork-termux
```

### Homebrew (Linux/macOS)

```bash
brew tap Dionanos/zork-termux
brew install zork-termux
```

## Launch

```bash
zork-termux
```
